## Instalace (Windows):
1. I am using python 3.11 (other versions should work too)
2. run "python -m venv venv" in this repository (it creates virtual environment)
3. run ".\venv\Scripts\activate" to activate virtual environment
4. run "python -m pip install -r .\requirements.txt" to install all required packages
5. run jupyter notebooks in some editor (Visual Studio Code) with created virtual environment
